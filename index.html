About

Welcome to NAχLA (North Africa x Linguistic Analysis), an independent group of researchers and a collaborative research project led by Fatma Ben Barka Messaoudi and Rayan Ziane. Our mission is to explore North African linguistic diversity, focusing on under-resourced languages, syntactic analysis, and sociolinguistic dynamics.

Projects

Balanced Corpus for Spoken Arabic in the Maghreb

This project aims to build a balanced corpus for spoken Arabic in the "Petit Maghreb," covering Tunisia, Algeria, and Morocco. We have already collected recordings for the tunisian and algerian variety, with transcription completed for the Tunisian part. The corpus is designed to support linguistic and computational analysis across regional dialects. Soon available in Cocoon.

Status: Ongoing

Contributors: Fatma Ben Barka Messaoudi, Anissa Fella Aissani, Rayan Ziane

Tunisian Morph-Based Treebank in Universal Dependencies

The goal of this project is to create a morphologically-based treebank for Tunisian Arabic. This treebank will adhere to the Universal Dependencies (UD) framework, nativally developped in Surface-Syntactic Universal Dependencies (SUD) for the morph-based version. It will serve as a valuable resource for syntax-based linguistic studies and computational modeling.

Status: Ongoing

Contributors: Rayan Ziane, Mustapha Khoudri, Fatma Ben Barka Messaoudi, Aya Gherab

ASR Model for Maghrebi Spoken Arabic (Latin Script)

This project focuses on developing an Automatic Speech Recognition (ASR) model to transcribe spoken Arabic from the Maghreb using a Latin-based script. The choice of Latin script is guided by user practices, particularly online, where the Arabizi writing style is widely used to phonetically represent Arabic. The model is designed to handle the unique phonetic and lexical challenges posed by Maghrebi Arabic dialects, paving the way for more accessible speech technology for under-resourced languages. Soon available in Huggingface.

Status: Ongoing

Contributors: Rayan Ziane
